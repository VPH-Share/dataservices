Test checklist
---------------------------------------------------------------------------------------

Treelist:
- long names croping
- expanded data tree shows tables with their columns and correct data types
- table click executes a SELECT * for that table
- column click executes a SELECT DISTINCT 'column' with its occurence in the datatset

Metadata:
- shows dataset metadata info ('unknown' entries if missing catalogue connection)

Download:
- Button disabled prior query execution
- Button enabled after query execution 
- XML/CSV results delivery after button click

Results table:
- field empty prior query execution
- results show after query execution in table
- Limit 1000 checkbox on per default
- On checkbox click a dialog shows for confirmation
- After confirmation query is executed without limiter

SQL Query:
- Execute button starts a custom query
